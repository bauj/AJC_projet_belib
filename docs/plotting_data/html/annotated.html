 CfLineData_sStructure associant un vecteur de donnees labelisé (float) à un LineStyle
 CFont_sStructure stockant les infos d'une police : chemin, taille et couleur
 CLineData_sStructure associant un vecteur de donnees labelisé (int) à un LineStyle
 CLineStyle_sStructure definissant un style de plot (ligne, point)
